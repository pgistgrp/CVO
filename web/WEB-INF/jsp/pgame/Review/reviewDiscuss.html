Review and Discuss: Concerns Synthesis Report
Purpose of this activity: In the previous stage we collected concerns about the transportation programming situation. The policy specialists have worked on these concerns and prepared this report, please take this opportunity to review this report and see if the report missed any important issues.
Instructions: At the beginning of each paragraph of the report, there is a mark "W", this is the entry for commenting on this paragraph. Click the mark "W" a new window will open to accept your comments on this paragraph. If following the mark "W" there is another mark "R", this means this paragraph has been commented, you can click on the "R" mark to read these comments.
