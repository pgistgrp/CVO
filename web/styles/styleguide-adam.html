Notes

HTML Elements

body

The body has a text-align:center declaration so that the #container DIV is centered in internet explorer. This declaration would cause the text of the page to be centered if a text-align:left declaration wasn't made in the #header and #container sections. The font is set to .8em Arial.

H2

Size 2em, and has no margin or padding. It inherits its color. Currently, it's being used to show the score in large letters in the voting tally box at the top of the page.

H3

The only change made to this style are that its height is set at 1.3em. H3 is being used for the titles of most sections.

Structural Elements

#colRight

Used for the "Add Concerns" box. Draws a 230px wide box with a padding of 5px and a margin-left of 15px.

#container

Wraps the entire document, except for the #header and the #headerMenu. It's currently set at a width of 950px, so that it will have some white space around the sides at 1024 resolution. It's got a text-align:left declaration to override the text centering in the body element. The #container is subdivided into two divs: (from to bottom) the #object, and the #discussion.

#header

No current styles. This DIV should be loaded with the contents of another file, containing the LIT logo, and the global search box.

#headerMenu

This div is the blue stripe at the top of the page.

#object

Contains the object. #object has a margin declaration.

#sortingMenu

Contains the sorting and filtering controls, and also the title of the discussion area (wrapped in H3 elements).

Global Classes

.clearBoth, .clearLeft, .clearRight

Used to apply the respective clearance declarations to div elements. .clearBoth is a class that is given the property clear:both in order to fix what appears to be a bug in Firefox that causes DIVs without height declarations to overlap.

.floatLeft, .floatRight

Used to give the respective float declarations to div elements.

Specific Classes

.button

Used for buttons throughout the page. Applies very narrow margins and padding. Does not contain a color (use one of the .box color schemes for that) or position (use one of the global classes for that).

.discussionBody

Contains the main text of a discussion item (e.g. the contents of a post or reply). No styles are currently applied.

discussionComments

Contains the comment count for a discussion item. Text in this div is right aligned.

discussion-left

A wrapper that contains discussionTitle, discussionBody, discussionComments, etc. It is 680px wide, with padding and a margin.

.headerButton

Used for buttons in the #headerMenu

.headerButtonCurrent

Used for the button in the #headerMenu that contains the current step in the LIT process. Has the same style as .headerButton.

.smallText

Used for small text, such as the text in the sorting menu, or in .buttons. Has a font size of .8em.

.tagsList

A wrapper for tags which sets list-style to "none", and sets padding and margin to 0.

.tagsInLine

Used for tags. Makes them float next to each other.

Link Styles

.headerButton a

.headerButtonCurrent a

#headerNext a

Applies a style to links in the "next step" button in the #headerMenu.
